# 🎮 MarvelWingspan Score

Aplicativo mobile simples para contagem de pontuação de partidas, inspirado no sistema do _Wingspan_ com mod da Marvel, com foco em praticidade durante o jogo e registro posterior.

---

## ✨ Funcionalidades

- 👥 Cadastro de jogadores
- 🎯 Criação de novas partidas
- 🧮 Contagem de pontuação por etapas:
  - Agentes
  - Operações
  - Bônus por rodada
  - Moedas
  - Recursos estocados
  - Capturas
  - Bônus por equipes
- 📊 Resultado final detalhado por jogador
- 🕓 Histórico de partidas salvas
- 📋 Exportação para planilha (copiar e colar direto no Excel)

---

## 📋 Formato de exportação

Pronto para colar diretamente no Excel ou Google Sheets.

---

## 📱 Tecnologias

- React Native
- Expo
- Expo Router
- AsyncStorage

---

## 🚀 Como rodar o projeto

```bash
npm install
npx expo start
```
